## Usage
Open the Cemu folder (*File > Open Cemu folder*), create a folder named `graphicPacks` if it doesn't already exist, and place the folders inside of it.  
Close and reopen Cemu, go to *Options > Graphic packs*, expand the list and enable the patch, then launch the game and test if it works. You can also enable and disable the patches while the game is running.  
Tested on Cemu 2.6.
